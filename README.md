<h2> SPA - приложение для работы с ресурсами для парсинга и сохранения/бновления нужной информации </h2>
 <p><b> 
 SPA - приложение для работы с ресурсами для парсинга и сохранения/обновления нужной информации.
 </b></p>
 <p>При запуске приложения открывается таблица источников, каждый источник имеет свои ссылки (url) по которым и осуществляется парсинг и сохранение информации.</p>
 <ul><b>Опции: </b>
 <li>Смена цветовой схемы (доступна светлая и темная, которая стоит по умолчанию)</li>
 <li>Сортировка таблицы источников по любому столбцу (в прямом и обратном направлении)</li>
 <li>Расширенный поиск в таблице источников по любым полям</li>
 <li>Создание, удаление и редактирование таблиц</li>
 <li></li>
 <li></li>
 <li></li>

</ul>
